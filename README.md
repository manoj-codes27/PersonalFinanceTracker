# Personal Finance Tracker

A simple and responsive Personal Finance Tracker built using HTML, Tailwind CSS, and Vanilla JavaScript. This application helps users manage their daily income and expenses, track their balance, and visualize spending with an interactive dashboard.

## Features

* Add income and expense transactions
* View current balance, total income, and total expenses
* Categorize transactions
* Search and filter transactions
* Interactive expense breakdown chart
* Delete individual transactions
* Export data as JSON
* Import data from JSON
* Local storage support (data persists in the browser)
* Responsive design for desktop and mobile
* Indian Rupee (₹) currency support

## Technologies Used

* HTML5
* Tailwind CSS
* Vanilla JavaScript
* LocalStorage API

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/manoj-codes27/PersonalFinanceTracker.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional installation or setup is required.
